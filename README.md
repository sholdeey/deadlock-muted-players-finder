<h1 align="center">Deadlock muted players finder</h1>
Extract SteamIDs from Deadlock voice_ban.dt to find your muted players.

``` To find file you need to open the main Steam folder then the userdata\[your profile]\1422450\voice_ban.dt ```\
Example: ```"C:\Program Files (x86)\Steam\userdata\1135604769\1422450\remote\voice_ban.dt"```
